# VoxelCraft Native

Native voxel-survival project built with Godot 4.6 and the Voxel Tools 1.6 C++ module.

## Download the playable Windows build

1. Open the repository **Actions** tab.
2. Open the latest **Build Windows EXE** run.
3. After the run is green, download the `VoxelCraftNative-Windows-x64` artifact.
4. Extract the downloaded ZIP.
5. Open `VoxelCraftNative.exe`.

You do not need to install Godot. GitHub Actions downloads the pinned custom engine and Windows export template, extracts the full project from `source/voxelcraft-native.zip`, and creates the executable automatically.

## Source

The complete Godot project is stored in `source/voxelcraft-native.zip`. It includes the GDScript source, scenes, original texture atlas, documentation, export preset, and license.

This is a clean-room voxel survival project and does not include Minecraft source code or Mojang assets.
